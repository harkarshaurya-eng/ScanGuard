# Sample Report Structure

This file is intentionally a structure example rather than a fabricated scan result.

Sections generated by `recon-ai report` include:

- Legal Authorization Reminder
- Executive Summary
- Scope
- Methodology
- Tools Used
- Asset Inventory
- Open Ports and Services
- Web Technologies and Reachable Assets
- Findings Table
- Detailed Findings
- Evidence and Raw Output Appendix
- Timeline
